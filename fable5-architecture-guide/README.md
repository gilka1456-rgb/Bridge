## 测试结果与差距

用四向生成图（举手姿势）测试，产出为一根**有横向条纹的模糊柱体**：无头部分离、无手臂、
轮廓波浪状。根因分析见 `01-result-diagnosis.md` —— 一句话：**包围盒归一化在举手/姿势
变化下失效，且视觉外壳天然无法处理四向姿势不一致。**

## 改进主线（一张图）

```
采集（活体扫描 / 照片导入）
  │  每视角：分割掩码 + 33 关键点
  ▼
[T4] 躺姿/斜姿 → 关键点主轴旋转进「身体直立坐标系」
  ▼
[T1] 关键点锚定归一化（骨盆定位 + 关键点身高定尺，替代包围盒）
  │        └─ [T3] 姿势一致性门（各视角关节角差异过大 → 拒收并引导）
  ▼
[T2] 双线性栅格化（消横纹）→ OrientationMask v3（带锚点）
  ▼
SDF 软雕刻（已有）──► 躯干/腿部宽度场
  ▼
[T5] 模板人体混合：关键点摆姿态的低模人体 + 外壳收缩包裹
  │        └─ 保证任何输入下都是「完整分明的人形」
  ▼
[T6] 柔和灵体材质（淡彩低发光 + 软菲涅尔 + 洋葱壳 + 脚部消散，默认关扫描线）
```
