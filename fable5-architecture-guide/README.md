## 当前差距

默认路径仍使用 `template-body.ts` 的 16 边径向管链，保证现有功能不受实验代码影响。
`ghost-body-v3=1` 已切换到单张连续水密人体，并能把标准 A-pose 烘焙回扫描姿势；端盖、分节
折面和肩胯硬切已经消失。`ghost-render-v3=1` 已启用共享透明渲染内核，解决前后表面双影和
透明深度错误。V4 已在同一人体上加入三级 LOD 和 GPU 姿势，自动质量切换仍等待 iPhone 数据。
V5 红灵/白灵与 V6 青色/紫色投影均已在共享结构上形成游戏级候选；当前等待视觉确认，再进入 V7 真机验收。
