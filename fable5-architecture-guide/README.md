## 当前差距

默认路径仍使用 `template-body.ts` 的 16 边径向管链，保证现有功能不受实验代码影响。
`ghost-body-v3=1` 已切换到单张连续水密人体，并能把标准 A-pose 烘焙回扫描姿势；端盖、分节
折面和肩胯硬切已经消失。当前尚未完成的是 V3 共享透明渲染内核，所以 V1/V2 截图仍沿用旧
材质，只用于验证几何与姿势，不代表最终奇幻或赛博画质。

## 改进主线（一张图）

```
采集（活体扫描 / 照片导入）
  │  每视角：分割掩码 + 33 关键点
  ▼
[T4] 躺姿/斜姿 → 关键点主轴旋转进「身体直立坐标系」
  ▼
[T1] 关键点锚定归一化（骨盆定位 + 关键点身高定尺，替代包围盒）
  │        └─ [T3] 姿势一致性门（各视角关节角差异过大 → 拒收并引导）
  ▼
[T2] 双线性栅格化（消横纹）→ OrientationMask v3（带锚点）
  ▼
SDF 软雕刻（已有）──► 躯干/腿部宽度场
  ▼
[T5] 模板人体混合：关键点摆姿态的低模人体 + 外壳收缩包裹
  │        └─ 保证任何输入下都是「完整分明的人形」
  ▼
[T6] 柔和灵体材质（淡彩低发光 + 软菲涅尔 + 洋葱壳 + 脚部消散，默认关扫描线）
```
