## 当前差距

四向生成图已经可以产出头、躯干、双臂和双腿完整的人形，但现有 `template-body.ts` 仍以
16 边径向管链分别生成头、躯干和四肢，再把它们装进同一个 `BufferGeometry`。几何在关节处
仍有独立端盖和低边数折面，截图看起来像几何人偶。第二阶段不再继续给这套管链叠材质，
而是替换为一张连续、水密、可分级的人体表面，再在同一表面上分支奇幻与赛博风格。

```
  ▼
[T2] 双线性栅格化（消横纹）→ OrientationMask v3（带锚点）
  ▼
SDF 软雕刻（已有）──► 躯干/腿部宽度场
  ▼
[T5] 模板人体混合：关键点摆姿态的低模人体 + 外壳收缩包裹

  ▼
[T6] 柔和灵体材质（淡彩低发光 + 软菲涅尔 + 洋葱壳 + 脚部消散，默认关扫描线）
```
